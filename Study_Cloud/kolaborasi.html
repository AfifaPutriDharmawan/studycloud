Kolaborasi Kelompok

Kerjakan tugas kelompok dengan mudah dan efisien

Berbagi Dokumen

Upload dan edit dokumen bersama anggota kelompok secara real-time. Dukung berbagai format file termasuk PDF, DOCX, dan PPT.

Diskusi Real-Time

Lakukan diskusi melalui chat kelompok dengan notifikasi instan. Bisa mention anggota tertentu menggunakan @nama.

Cara Membuat Kelompok Baru

  1. Klik menu "Kolaborasi" di navbar
  2. Pilih "Buat Kelompok Baru"
  3. Isi nama kelompok dan pilih mata kuliah
  4. Tambahkan anggota dengan mencari nama atau NIM
  5. Klik "Buat Kelompok" untuk menyimpan
